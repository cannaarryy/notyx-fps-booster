# notyx-fps-booster
Menú optimizador de FPS. Cambia cielo, iluminación y efectos visuales sin tocar mapa, tráfico ni objetos. Presets: ULTRA BOOST (máxima limpieza), RAPID BOOST (equilibrio), LOW (PCs modestos) y ORIGINAL. Filtros especiales: Noche Eterna, Niebla Densa, Blanco y Negro, Calor, Viñeta... Botón RESET instantáneo y FPS en tiempo real. Standalone, ligero
